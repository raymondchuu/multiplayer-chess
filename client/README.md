## This contains the front-end code to my online chess game
The front end was designed in ReactJS and contains all of the logic behind the game.

## To Get Started
1. Fork this repository, then clone your fork of this repository.
2. Install dependencies using the npm install command.
3. Start the web server using the npm run dev command. The app will be served at http://localhost:3000/.
4. Go to http://localhost:3000/ in your browser.

## Dependencies
- react-router-dom
- react-scroll-to-bottom
- simple-peer
- socket.io-client
- use-sound
- uuid
- web-vitals
